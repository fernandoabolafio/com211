  • Condução de entrevistas
  • Verificação de métricas definidas

Nota: As métricas dos processos quantificam atributos como tempo, esforço, número de incidências, dentre outros

Produtos: são os resultados (programas, projetos, código fonte, casos de teste) da execução do processo. As medidas incluem o tamanho do produto, linhas de código, a complexidade. Os recursos são conjunto de elementos que são entradas para a produção do software. As medidas incluem a eficiência de uma ferramenta para identificar Bugs, a produtividade de um perfil de programadores, entre outros.
